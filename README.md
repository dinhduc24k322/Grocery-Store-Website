# Trang web cửa hàng tạp hóa
## Mục đích
- Trang web bán tạp hóa trên máy tính
- Hiển thị các sản phẩm
- Chức năng tìm kiếm
- Chức năng mua các sản phẩm
- Chức năng giỏ hàng

## Chức năng phát triển 
- giảm giá các mặt hàng
- Quảng cáo các sản phẩm
- chức năng thanh toán online
